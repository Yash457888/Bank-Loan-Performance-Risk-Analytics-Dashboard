**📊 Bank Loan Performance Analysis & Dashboard**

**🔷 Overview**
This project focuses on analysing a bank’s loan portfolio to understand lending performance, borrower risk, and repayment trends. The goal is to build a data-driven reporting system that helps track key financial metrics and supports better decision-making in lending operations.

**🔷 Dataset**
The dataset contains detailed information about loan applications, borrower profiles, and repayment behaviour. Key fields include:

Loan Amount, Interest Rate, and Instalment
Loan Status (Fully Paid, Current, Charged Off)
Borrower details (Income, Employment Length, Home Ownership)
Issue Date and Payment data

**👉These variables are used to evaluate loan performance and credit risk.**

**🔷 Tools & Technologies**
SQL → Data extraction, KPI calculation, validation
Power BI → Dashboard creation and data visualization

**🔷 Project Steps**
**Understanding Business Problem**
Defined KPIs like loan applications, funded amount, repayment, interest rate, and DTI.
**SQL Data Analysis**
Used aggregate functions (COUNT, SUM, AVG)
Performed MTD and MoM analysis
Segmented data by loan status, state, purpose, etc.
Validated all dashboard metrics using SQL queries
**Data Visualization (Power BI)**
Built interactive dashboards to represent trends, distributions, and performance.
**Reporting & Presentation**
Created a structured report and PPT to communicate insights clearly.

**🔷 Dashboard**
**The project includes 3 main dashboards:**

**1. Summary Dashboard**
Total Loan Applications, Funded Amount, Amount Received
Average Interest Rate & DTI
Good vs Bad Loan analysis
**👉 As shown in the dashboard (page 6), KPIs like ~38K applications and loan quality segmentation are highlighted clearly**

**2. Overview Dashboard**
Monthly trends (loan applications, funding, repayments)
Regional analysis by state
Loan segmentation (term, purpose, employment length, home ownership)
**👉 Visuals like line charts and donut charts help identify trends and distribution patterns (page 9)**

**3. Details Dashboard**
Granular, record-level loan data
Filter-based analysis for deeper insights
**👉 Provides a complete view of borrower and loan-level data (page 11)**

**🔷 Results & Insights**
Majority of loans fall under good loan category (Fully Paid & Current), indicating stable portfolio performance
A smaller portion of bad loans (Charged Off) highlights risk exposure
Loan distribution varies significantly across purpose, term, and borrower profile
Metrics like DTI and interest rate help assess borrower risk and repayment capacity

**🔷 Key Learnings**
Translating business requirements into measurable KPIs
Writing optimized SQL queries for real-world datasets
Building interactive dashboards in Power BI
Understanding financial metrics like loan risk, DTI, and portfolio health

**🔷 Conclusion**
This project demonstrates how data analytics can be applied in the banking domain to monitor loan performance, assess risk, and support strategic decision-making using SQL and Power BI.
